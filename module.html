Python 模块学习

模块简介

在 Python 中，模块是一个包含 Python 定义和语句的文件，帮助将代码组织成可管理的部分。使用模块可以带来以下好处：

  • 代码组织和重用性
  • 命名空间管理
  • 实现隐藏
  • 跨多个项目共享代码

导入模块的方式

# 导入整个模块
import math

# 从模块中导入特定项目
from math import sqrt, pi

# 从模块中导入所有项目（不推荐）
from math import *

# 使用别名导入
import math as m

🧠 小提示

  • Python 中，模块需要使用 import 模块名 语句导入。
  • random 模块里常用的函数 randint(a, b) 会返回 a 到 b（包含 a 和 b） 的随机整数。
  • 函数调用格式：模块名.函数名(参数)。

Python 标准库

Python 标准库包含许多有用的模块和工具，随 Python 一起安装。常见模块有：

  • math - 数学函数
  • random - 随机数生成
  • datetime - 日期和时间
  • os - 操作系统接口
  • sys - 系统参数和函数
  • json - JSON 数据处理
  • re - 正则表达式
  • collections - 特殊容器数据类型

使用标准库可以大大提高开发效率，减少"重新发明轮子"的情况。在编写新代码前，建议先查看标准库中是否已有解决方案。示例：使用 datetime 模块获取当前日期和时间

import datetime

current_time = datetime.datetime.now()
print(f"当前日期和时间: {current_time}")

formatted_date = current_time.strftime("%Y-%m-%d %H:%M:%S")
print(f"格式化后: {formatted_date}")
