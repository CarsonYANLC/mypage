回归 (Regression)、分类 (Classification)、聚类 (Clustering) & 线性回归 (Linear Regression)、逻辑回归 (Logistic Regression)

一、回归、分类、聚类

1. 回归 (Regression)

定义：回归的目标是预测一个 连续值，例如房价、温度、销售量等。

常见算法：

  • 线性回归 (Linear Regression)
  • 多项式回归 (Polynomial Regression)
  • 回归树 (Regression Tree)

案例：房价预测、股票价格预测、销售量预测等。

2. 分类 (Classification)

定义：分类的目标是预测一个 离散类别标签，如垃圾邮件或正常邮件、图像中的猫/狗识别等。

常见算法：

  • 逻辑回归 (Logistic Regression)
  • 决策树 (Decision Tree)
  • 随机森林 (Random Forest)
  • 支持向量机 (SVM)
  • 神经网络 (ANN)

案例：垃圾邮件过滤、肿瘤良恶性判别、手写数字识别等。

3. 聚类 (Clustering)

定义：将相似的数据样本划分到同一组，没有明确标签，属于无监督学习，用于数据探索或结构发现。

常见算法：

  • K-means
  • 层次聚类 (Hierarchical Clustering)
  • DBSCAN

案例：客户分群、图像分割、文本聚类等。

方法与注意点

  • 回归、分类都属于 监督学习（有标签）；聚类属于 无监督学习（无标签）。
  • 如果预测目标是连续值 → 选择回归；如果是有限类别 → 选择分类；如果要发现未知类别或相似分组 → 选择聚类。

二、线性回归 & 逻辑回归

2.1 线性回归 (Linear Regression)

基本思想：假设输出 y 与输入特征 x 之间存在近似线性关系，常用 最小二乘法(OLS) 来拟合参数。

模型形式：

y = w0 + w1*x1 + w2*x2 + ... + wn*xn

参数求解：最常见方式是最小化平方误差 (RSS)。
RSS = Σ (y_i - y_pred_i)²。

适用场景：目标值是连续变量，且与特征大体呈线性关系。

2.2 逻辑回归 (Logistic Regression)

基本思想：主要用于二分类或多分类，通过 Sigmoid 函数 将线性组合的结果映射到 (0,1) 之间，输出“某类别”的概率。

模型形式：

p = 1 / (1 + e^(-z)), 其中 z = w0 + w1*x1 + ... + wn*xn

损失函数：常用对数似然 (Log-likelihood)，通过极大似然估计求解参数。

应用场景：二分类(如垃圾邮件/正常邮件)，也可用 One-vs-Rest 扩展到多分类。

方法与案例

  • 线性回归案例：房价预测、身高和体重关系、销售额预测等。
  • 逻辑回归案例：垃圾邮件检测、用户流失预测、肿瘤良恶性判别（二分类）。

判断题 (共 6 题)

15. 判断：回归和聚类都需要标签(y)才能训练模型。

正确答案：False

解析：回归需要标签(连续值)，但聚类是无监督学习，没有标签。

16. 判断：线性回归只能用于一元输入特征，无法扩展到多元输入。

正确答案：False

解析：线性回归可以扩展到多元输入特征 (多元线性回归)。

17. 判断：逻辑回归可以用来处理多分类场景，但需采取 One-vs-Rest 或 Softmax 等方式。

正确答案：True

解析：逻辑回归最常用于二分类，但可以通过多分类策略扩展到多分类问题，如 One-vs-Rest (OvR) 或多项式Logistic(Softmax)。

18. 判断：如果目标是预测股票是否会上涨或下跌，最好把它看作回归问题。

正确答案：False

解析：上涨/下跌是二分类问题，属于离散标签。除非你要预测确切的涨幅值，才是回归。

19. 判断：聚类可以用于处理没有标签的数据集。

正确答案：True

解析：聚类是无监督学习，一般用于没有标签的数据。

20. 判断：在多项式回归中，如果将输入 x 转化为 (x, x², x³, ...) 就可以在某种程度上拟合非线性关系。

正确答案：True

解析：多项式回归是通过增加 x 的幂次项来捕捉一定程度的非线性关系。
